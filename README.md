# Velaan360
AI-Based Multilingual Voice-Enabled Agricultural Support System

An intelligent AgriTech solution that provides farmers with multilingual voice-based assistance, real-time weather risk alerts, and offline support. The system leverages AI, NLP, speech recognition, and weather APIs to deliver personalized crop guidance, pest control recommendations, and agricultural information in regional languages.

## 🚀 Features

* 🎙️ Multilingual Voice Input and Output
* 🌦️ Real-Time Weather Risk Alerts
* 🌱 Crop and Fertilizer Recommendations
* 🐛 Pest and Disease Advisory
* 📶 Offline Query Support
* 🗣️ Text-to-Speech Responses
* 📈 AI and Rule-Based Decision Making

## 🛠️ Technologies Used

* Python
* Vosk Speech Recognition
* gTTS (Google Text-to-Speech)
* spaCy NLP
* Hugging Face MarianMT
* SQLite
* OpenWeatherMap API
* Tkinter

## 🌍 Supported Languages

* English
* Tamil
* Hindi

## 🎯 Objective

To provide accessible, voice-driven, and localized agricultural assistance for farmers, especially in rural areas with limited internet connectivity.
